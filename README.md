## 📖 Overview

**ChronosPulse-AI** is an AI-powered healthcare platform designed to reduce hospital waiting times through intelligent symptom assessment and live patient queue management.

Patients enter their symptoms, and the system performs an AI-assisted preliminary urgency assessment. Based on the estimated priority, the platform provides a live queue position, estimated waiting time, and real-time updates, helping hospitals improve efficiency while enhancing the patient experience.

> **Note:** ChronosPulse-AI is intended for educational, research, and demonstration purposes. It is **not a replacement for professional medical diagnosis or emergency healthcare services.**

# 🚀 Installation

## Clone the repository

```bash
git clone https://github.com/kartik47-ai/ChronosPulse-AI.git
```

## Navigate into the project

```bash
cd ChronosPulse-AI
```

## Install backend dependencies

```bash
pip install -r backend/requirements.txt
```

## Start the FastAPI server

```bash
uvicorn backend.main:app --reload
```

## Open the frontend

Launch `frontend/index.html` in your browser or serve it using your preferred local server.

# ⚙️ How It Works

1. Patient enters their symptoms.
2. The AI performs an initial symptom assessment.
3. The system estimates the patient's urgency level.
4. The patient is assigned a queue position.
5. A live countdown displays the estimated waiting time.
6. Queue updates occur in real time as patients are served.
